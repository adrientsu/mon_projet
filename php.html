🐘 Leçon 1: Introduction à PHP

Qu'est-ce que PHP ?

PHP (Hypertext Preprocessor) est un langage de programmation côté serveur, spécialement conçu pour le développement web. Il permet de créer des pages web dynamiques en générant du contenu HTML à la volée.

💡 À savoir

PHP est exécuté sur le serveur, pas dans le navigateur. Le code PHP est invisible pour l'utilisateur, qui ne voit que le HTML généré.

Balises PHP

Le code PHP doit être placé entre des balises spéciales :

Structure de base PHP 
<?php
    // Ceci est un commentaire PHP
    echo "Hello World !";
    print "Bonjour le monde !";
?>

Fichier PHP typique

fichier.php 
<?php
// Configuration
$nom_site = "Mon Site Web";
$annee = 2024;

// Calculs
$age = 25;
$age_dans_10_ans = $age + 10;
?>

<!DOCTYPE html>
<html>
<head>
    <title><?php echo $nom_site; ?></title>
</head>
<body>
    <h1>Bienvenue sur <?= $nom_site ?></h1>
    <p>Vous avez <?= $age ?> ans</p>
    <p>Dans 10 ans, vous aurez <?= $age_dans_10_ans ?> ans</p>
</body>
</html>
Résultat (côté serveur) :
<!DOCTYPE html>
<html>
<head>
  <title>Mon Site Web</title>
</head>
<body>
  <h1>Bienvenue sur Mon Site Web</h1>
  <p>Vous avez 25 ans</p>
  <p>Dans 10 ans, vous aurez 35 ans</p>
</body>
</html>
💡 Astuce

<?= $variable ?> est un raccourci pour <?php echo $variable; ?>

⚠️ Important

PHP nécessite un serveur web (comme Apache ou Nginx) avec PHP installé. On ne peut pas exécuter PHP en ouvrant simplement un fichier .html dans le navigateur.
