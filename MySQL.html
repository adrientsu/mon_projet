Chapitre 1: Introduction à MySQL Débutant

Qu'est-ce que MySQL ?

Système de gestion de bases de données relationnelles

MySQL est un système de gestion de bases de données relationnelles (SGBDR) open-source. C'est l'un des systèmes les plus populaires au monde, utilisé par des entreprises comme Facebook, Twitter, YouTube, etc.

Caractéristiques

  • Relationnel : données organisées en tables reliées entre elles
  • SQL : langage de requête structuré
  • ACID : Atomicité, Cohérence, Isolation, Durabilité
  • Multi-utilisateurs : gestion des droits d'accès
  • Performant : optimisé pour la lecture/écriture
  • Multi-plateforme : Windows, Linux, macOS
