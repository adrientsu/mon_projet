Chapitre 1: Introduction à Bootstrap

Qu'est-ce que Bootstrap ?

Le framework CSS le plus populaire au monde

Bootstrap est un framework CSS gratuit et open-source créé par Twitter. Il permet de créer rapidement des sites web responsives et modernes.

Caractéristiques

  • Responsive : s'adapte à tous les écrans
  • Mobile-first : conçu d'abord pour mobile
  • Composants prêts à l'emploi : navbar, cards, modals...
  • Système de grille : 12 colonnes
  • Utilitaires : classes pour tout (spacing, couleurs...)

Installation

CDN (recommandé)
<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
Installation locale
# npm
npm install bootstrap@5.3.2

# yarn
yarn add bootstrap@5.3.2

Structure de base

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ma page Bootstrap</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container">
        <h1>Bonjour Bootstrap !</h1>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Chapitre 2: Système de grille

Les bases de la grille Bootstrap

12 colonnes responsives

Bootstrap utilise un système de grille basé sur flexbox avec 12 colonnes.

Conteneurs

Classe Description
.container Largeur fixe, centré, responsive
.container-fluid Largeur 100% de l'écran

Points de rupture (breakpoints)

Breakpoint Classe Largeur
XScol-< 576px
SMcol-sm-≥ 576px
MDcol-md-≥ 768px
LGcol-lg-≥ 992px
XLcol-xl-≥ 1200px
XXLcol-xxl-≥ 1400px

Exemples de grille

<!-- 3 colonnes égales -->
<div class="container">
    <div class="row">
        <div class="col">Colonne 1</div>
        <div class="col">Colonne 2</div>
        <div class="col">Colonne 3</div>
    </div>
</div>

<!-- Colonnes de tailles différentes -->
<div class="container">
    <div class="row">
        <div class="col-8">8 colonnes</div>
        <div class="col-4">4 colonnes</div>
    </div>
</div>
