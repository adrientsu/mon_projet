Chapitre 1: Introduction à C++ Débutant

Qu'est-ce que C++ ?

Histoire, philosophie et utilisations

Le C++ est un langage de programmation compilé créé par Bjarne Stroustrup en 1983 chez Bell Labs. Il est une extension du langage C avec l'ajout de la programmation orientée objet.

Caractéristiques principales

  • Compilé : code source transformé en code machine
  • Performant : très proche du matériel
  • Multi-paradigme : procédural, orienté objet, générique, fonctionnel
  • Typage statique : les types sont vérifiés à la compilation
  • Gestion manuelle de la mémoire : contrôle total
  • Multi-plateforme : Windows, Linux, macOS, embarqué

Domaines d'application

  • Jeux vidéo (Unreal Engine)
  • Systèmes embarqués
  • Bases de données
  • Robotique
  • Trading haute fréquence
