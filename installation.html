Installation WAMP et XAMPP

Guides complets pour installer et configurer vos environnements de développement web

🌐 Qu'est-ce qu'un serveur local ?

Un serveur local (WAMP, XAMPP, MAMP, LAMP) est un environnement de développement qui simule un serveur web sur votre ordinateur. Il permet de développer et tester des sites web dynamiques (PHP, MySQL) sans avoir besoin d'un hébergement en ligne.

Composants communs

  • Apache : Serveur web HTTP
  • PHP : Langage de programmation côté serveur
  • MySQL / MariaDB : Système de gestion de base de données
  • phpMyAdmin : Interface web pour gérer les bases de données

📦 WAMP (Windows, Apache, MySQL, PHP)

🪟 Spécifique Windows

WAMP est conçu exclusivement pour Windows. Il s'intègre parfaitement avec le système d'exploitation Microsoft.

Présentation

WAMP est un environnement de développement pour Windows qui permet de faire tourner localement un serveur web Apache, une base de données MySQL et le langage PHP.

Caractéristique Détail
Dernière version WAMP 3.3.0 (64-bit)
PHP inclus PHP 8.2, 8.1, 8.0, 7.4, 7.3, 7.2
MySQL/MariaDB MariaDB 10.11, 10.10, 10.9, 10.8, 10.6, 10.5, 10.4, 10.3, 10.2
MySQL 8.0, 5.7
Apache Apache 2.4
Interface Icône dans la barre des tâches + interface web

📥 Installation de WAMP (étape par étape)

1

Téléchargement

Rendez-vous sur le site officiel : https://www.wampserver.com/

Choisissez la version adaptée à votre système (32-bit ou 64-bit).

💡 Vérifiez votre architecture : Faites un clic droit sur "Ce PC" → Propriétés pour voir si votre Windows est en 32 ou 64 bits.
2

Prérequis : Visual C++ Redistributable

WAMP nécessite les paquets Visual C++ Redistributable. Si vous ne les avez pas, l'installateur vous proposera de les télécharger.

Paquets nécessaires
- Visual C++ Redistributable for Visual Studio 2012
- Visual C++ Redistributable for Visual Studio 2013
- Visual C++ Redistributable for Visual Studio 2015-2022 (x64)
3

Lancer l'installation

Exécutez le fichier téléchargé (wampserver.exe).

  • Acceptez la licence
  • Choisissez le dossier d'installation (par défaut : C:\wamp64)
  • Choisissez votre navigateur par défaut (Firefox, Chrome, etc.)
  • Choisissez votre éditeur de texte par défaut (Notepad++, VS Code, etc.)
