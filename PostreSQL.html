🐘 Leçon 1: Introduction aux bases de données

Qu'est-ce qu'une base de données ?

Une base de données est un système organisé pour stocker, gérer et récupérer des données de manière efficace. PostgreSQL est un système de gestion de base de données relationnelle (SGBDR) open-source très puissant.

💡 Pourquoi PostgreSQL ?
  • Open-source et gratuit
  • Conforme aux standards SQL
  • Performant et extensible
  • Supporte JSON et les données non-structurées
  • Transactions ACID (Atomicité, Cohérence, Isolation, Durabilité)

Structure d'une base de données relationnelle

Une base de données relationnelle organise les données en tables, composées de lignes et de colonnes.

Exemple de table "utilisateurs" :
id (PK) nom email age ville
1 Dupont jean@email.com 25 Paris
2 Martin sophie@email.com 30 Lyon
3 Bernard pierre@email.com 22 Marseille

Vocabulaire SQL

Terme Description

Table Structure organisée en lignes et colonnes
Colonne (champ) Attribut d'une table
Ligne (enregistrement) Instance de données
Clé primaire (PK) Identifiant unique d'une ligne
Clé étrangère (FK) Référence vers une autre table
