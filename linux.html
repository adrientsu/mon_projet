200 Commandes Linux Essentielles

Guide complet avec exemples et explications détaillées pour maîtriser Linux

🐧 À propos de ce guide

Ce guide présente 200 commandes Linux essentielles classées par catégorie. Chaque commande inclut une description, un exemple concret et des explications. Que vous soyez débutant ou administrateur expérimenté, ces commandes couvrent tous les aspects de l'utilisation quotidienne de Linux.

💡 Astuce

Pour chaque commande, vous pouvez obtenir plus d'informations avec :

$ man nom_commande
$ nom_commande --help

1️⃣ Navigation et système - Commandes 1 à 10

# Commande Description Exemple
1 pwd Affiche le répertoire courant (Print Working Directory) pwd → /home/utilisateur
2 cd Change de répertoire (Change Directory) cd /var/log
3 cd .. Remonte au répertoire parent cd ..
4 cd ~ Va dans le répertoire personnel de l'utilisateur cd ~ ou cd
5 cd - Retourne au répertoire précédent cd -
6 ls Liste les fichiers et dossiers (List) ls
7 ls -l Liste détaillée avec permissions, taille, date ls -l
8 ls -a Liste tous les fichiers, y compris les cachés (commençant par .) ls -a
9 ls -la Liste détaillée de tous les fichiers (équivalent à ls -l -a) ls -la
