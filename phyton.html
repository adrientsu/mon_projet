Chapitre 1: Introduction à Python Débutant

Qu'est-ce que Python ?

Histoire, philosophie et utilisations

Python est un langage de programmation interprété, créé par Guido van Rossum en 1991. Son nom vient de la série télévisée "Monty Python's Flying Circus".

Caractéristiques principales

  • Interprété : pas besoin de compilation
  • Orienté objet : tout est objet
  • Typage dynamique : pas de déclaration de types
  • Multi-paradigme : impératif, fonctionnel, objet
  • Multi-plateforme : Windows, Mac, Linux
  • Grande bibliothèque standard : "batteries included"

Domaines d'application

  • Développement web (Django, Flask)
  • Data science (Pandas, NumPy)
  • Intelligence artificielle (TensorFlow)
  • Automatisation (scripts)
  • Jeux (Pygame)
  • Applications mobiles (Kivy)
