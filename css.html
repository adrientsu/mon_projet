🎨 Leçon 1: Introduction à CSS

Qu'est-ce que le CSS ?

CSS (Cascading Style Sheets) est le langage qui permet de styliser les pages HTML. Il contrôle l'apparence des éléments : couleurs, polices, espacements, positions, animations, etc.

💡 À savoir

CSS signifie "Cascading Style Sheets" (Feuilles de style en cascade). Le mot "cascade" est important car les styles s'appliquent selon un ordre de priorité.

À quoi sert CSS ?

  • Mise en forme : couleurs, polices, tailles
  • Mise en page : positionnement des éléments
  • Responsive design : adaptation aux écrans
  • Animations : effets et transitions
  • Amélioration de l'expérience utilisateur
